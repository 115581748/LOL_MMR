# 峡谷天平

一个用动态机制图解释《英雄联盟》排位系统中 MMR、Rank、LP、匹配与常见 Elo 叙事的中文单页网站。

直接打开 `index.html` 即可浏览，无需安装依赖。

峡谷交互模型使用 Riot Data Dragon 16.12.1 的英雄头像、小地图和召唤师技能图标。`assets/rift-atmosphere.png` 为本项目生成的氛围图。

## 内容边界

- Riot 官方材料用于确认 MMR、Rank 与 LP 的公开机制。
- Glicko、TrueSkill2、Elo-MMR 论文只用于解释现代评分系统的一般思想，不声称 Riot 使用这些具体算法。
- 本项目并非 Riot Games 官方项目。Riot Games 及《英雄联盟》相关素材归其各自权利人所有，使用时应遵守 Riot 的玩家内容政策。
